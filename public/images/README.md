# 图片资源说明

本目录存放网站所需的所有图片资源。

## 📁 目录结构

```
images/
├── banner/          # Banner轮播图片
├── gallery/         # 企业风采图片
├── products/        # 产品展示图片
└── README.md        # 本说明文档
```

## 🖼️ 图片规格要求

### Banner轮播图（banner/）
- **数量**：4张
- **命名**：banner-1.jpg ~ banner-4.jpg
- **推荐尺寸**：2070 x 1380 像素（或16:9比例）
- **格式**：JPG/PNG
- **文件大小**：建议每张 < 500KB
- **内容建议**：
  - banner-1.jpg: 现代化工厂车间全景
  - banner-2.jpg: 精密加工设备特写
  - banner-3.jpg: 自动化生产线场景
  - banner-4.jpg: 智能制造车间实景

### 企业风采图（gallery/）
- **数量**：6张
- **命名**：gallery-1.jpg ~ gallery-6.jpg
- **推荐尺寸**：800 x 600 像素（或4:3比例）
- **格式**：JPG/PNG
- **文件大小**：建议每张 < 300KB
- **内容建议**：
  - gallery-1.jpg: 现代化厂房外观
  - gallery-2.jpg: 智能生产车间
  - gallery-3.jpg: 质检中心场景
  - gallery-4.jpg: 仓储物流区域
  - gallery-5.jpg: 研发团队工作场景
  - gallery-6.jpg: 企业荣誉证书展示

### 产品展示图（products/）
- **数量**：6张
- **命名**：product-1.jpg ~ product-6.jpg
- **推荐尺寸**：800 x 800 像素（或1:1比例）
- **格式**：JPG/PNG
- **文件大小**：建议每张 < 200KB
- **内容建议**：
  - product-1.jpg: 现代简约风格茶几
  - product-2.jpg: 工业风金属茶几
  - product-3.jpg: 北欧风茶几
  - product-4.jpg: 轻奢风茶几
  - product-5.jpg: 创意设计茶几
  - product-6.jpg: 多功能茶几

## 🎨 图片优化建议

1. **压缩优化**：使用TinyPNG或ImageOptim压缩图片
2. **格式选择**：照片使用JPG，图标使用PNG
3. **响应式**：可准备@2x版本用于高清屏
4. **懒加载**：代码已配置图片懒加载，无需额外处理

## 🔄 替换图片步骤

1. 准备符合规格的图片
2. 按照命名规则重命名图片
3. 将图片放入对应目录
4. 刷新浏览器查看效果

## 📝 注意事项

- ⚠️ 请确保图片文件名完全匹配（区分大小写）
- ⚠️ 建议使用JPG格式以减小文件体积
- ⚠️ 图片内容应符合企业形象和产品特点
- ⚠️ 避免使用过大的图片影响加载速度
